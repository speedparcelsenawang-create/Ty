# PrimeReact Editor Demo

Demo aplikasi menggunakan PrimeReact Editor component (v9).

## Fitur

- Basic Editor dengan toolbar lengkap
- Custom Toolbar Editor dengan konfigurasi khusus
- Output HTML real-time
- Responsive design

## Cara Menjalankan

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Buka browser ke `http://localhost:3000`

## Teknologi yang Digunakan

- React 18
- PrimeReact v9
- Vite
- Quill Editor

## Struktur Project

- `src/App.jsx` - Main component dengan Editor examples
- `src/main.jsx` - Entry point aplikasi
- `src/App.css` - Styling untuk aplikasi
- `index.html` - HTML template
- `vite.config.js` - Vite configuration

## PrimeReact Editor Features

Editor ini berbasis Quill dan menyediakan:
- Rich text editing
- Custom toolbar
- HTML output
- Berbagai format text (bold, italic, underline, lists, dll)
- Support untuk images, links, dan code blocks
